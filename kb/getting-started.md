# Getting Started Guide

## Account Setup

1. Sign up at https://app.example.com/signup
2. Verify your email address (check spam folder if not received)
3. Complete your profile in Settings > Profile
4. Enable two-factor authentication for security

## Creating Your First Workspace

1. Click "New Workspace" from the dashboard
2. Choose a name (can be changed later)
3. Select your workspace type: Personal, Team, or Organization
4. Invite team members via email

## Inviting Team Members

- Basic plan: Up to 5 team members
- Pro plan: Up to 25 team members
- Enterprise plan: Unlimited team members
- Roles available: Admin, Editor, Viewer

## Uploading Files

1. Drag and drop files into any workspace
2. Or click "Upload" and select files
3. Supported formats: PDF, Office documents, images, videos
4. Maximum file size: 5GB per file

## Sharing Content

- Share with team: Click "Share" and select team members
- Share externally: Generate a public link (Pro and Enterprise)
- Link expiration: Set links to expire after 1, 7, or 30 days
- Password protection: Add a password to shared links

## Mobile Access

1. Download the app from App Store or Google Play
2. Sign in with your account credentials
3. Enable biometric login for quick access
4. Enable offline mode to access files without internet

## Getting Help

- In-app help: Click the "?" icon in the bottom right
- Documentation: https://docs.example.com
- Email support: support@example.com
- Live chat: Available 9 AM - 5 PM EST (Pro and Enterprise)
