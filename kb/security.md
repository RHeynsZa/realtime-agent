# Security & Privacy

## Data Protection

- All data encrypted at rest using AES-256
- All data encrypted in transit using TLS 1.3
- Data stored in SOC 2 compliant data centers
- Daily backups with 30-day retention

## Compliance

- SOC 2 Type II certified
- GDPR compliant
- CCPA compliant
- HIPAA compliant (Enterprise plan with BAA)

## Authentication

- Password requirements: minimum 12 characters, 1 uppercase, 1 number, 1 special character
- Two-factor authentication (2FA) available for all accounts
- Single Sign-On (SSO) via SAML 2.0 (Enterprise only)
- Session timeout: 24 hours of inactivity

## Access Control

- Role-based access control (RBAC)
- Audit logs retained for 1 year
- IP allowlisting available on Enterprise plan
- Device management and remote wipe (Enterprise only)

## Incident Response

- 24/7 security monitoring
- Security incidents reported within 72 hours
- Bug bounty program available at security@example.com
- Annual third-party penetration testing

## Data Deletion

- Account data deleted within 30 days of account closure
- Right to erasure (GDPR) requests processed within 30 days
- Backup data purged within 90 days of deletion request
