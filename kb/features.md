# Product Features

## Core Features

- Real-time collaboration: Up to 50 users per workspace
- Cloud storage: 10GB (Basic), 100GB (Pro), Unlimited (Enterprise)
- File sharing: Share files up to 5GB each
- Version history: 30 days (Basic), 90 days (Pro), Unlimited (Enterprise)

## Security Features

- Two-factor authentication (2FA)
- Single Sign-On (SSO) - Enterprise only
- End-to-end encryption for all data
- SOC 2 Type II certified

## Integrations

- Slack integration
- Microsoft Teams integration
- Google Workspace integration
- Zapier (500+ apps)
- REST API access (Pro and Enterprise)

## Mobile Apps

- iOS app: Requires iOS 14.0 or later
- Android app: Requires Android 8.0 or later
- Offline mode available on Pro and Enterprise plans

## Analytics

- Basic analytics: Page views and user activity
- Advanced analytics (Pro): Custom reports, export to CSV
- Enterprise analytics: Real-time dashboards, API access
